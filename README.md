
# 🏥 Hospital Emergency Room Dashboard

An interactive Excel dashboard built to analyze Hospital Emergency Room (ER) 
patient data — created as part of my Data Analyst Internship at 
**Analytics Career Connect**.

## 📊 Dashboard Preview
![Dashboard Screenshot](dashboard_screenshot.png)

## 🎯 Objective
To analyze ER patient data and provide actionable insights on patient flow, 
wait times, satisfaction, and department referrals to help improve hospital 
efficiency and service quality.

## 🔑 Key Metrics Analyzed
- **Patient Admission Status** — Admitted vs Not Admitted (with % breakdown)
- **Patient Age Distribution** — patients grouped by age brackets (0-9 to 70-79)
- **Timeliness** — % of patients seen within 30 minutes (Delay vs Ontime)
- **Gender Analysis** — patient distribution by gender
- **Department Referrals** — most frequently referred departments 
  (Cardiology, Orthopedics, Neurology, General Practice, etc.)

## 📈 KPI Cards (with Sparklines)
- **Number of Patients** — daily trend via area sparkline to spot busy days
- **Average Wait Time** — area sparkline to flag days needing improvement
- **Patient Satisfaction Score** — daily trend to spot drops linked to busy periods

## 🛠️ Tools Used
- Microsoft Excel (PivotTables, PivotCharts, Slicers, Sparklines)
- Python (pandas, openpyxl) — for data generation/processing
